# Study Plant Vue App

这是一个基于Vue 3和TypeScript的项目，使用Vite构建工具。

## 项目设置

```sh
npm install
```

### 编译和热重载

```sh
npm run dev
```

### 类型检查、编译和最小化生产环境

```sh
npm run build
```

### 预览生产环境构建

```sh
npm run preview
```

### Lint检查

```sh
npm run lint
```

## Git使用

项目已初始化Git，你可以直接进行提交和推送：

```sh
git add .
git commit -m "Initial commit"
git remote add origin <你的远程仓库地址>
git push -u origin main
```
